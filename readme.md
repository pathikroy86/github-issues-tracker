1. The main difference between var, let and const is in var and let we can reassign the value, but in const variable reassigning is not possible. In some occasions var can be accessed from outside of a scope, but let and const are block scope variable type. They can't be accessed outside any scope.

2. Spread operator (...) is something by which we can copy an array to another new array without copying the reference.

3. In map(), it loops through the whole array and returns a new array. On the otherhand, filter() loops through the whole array, finds the elements satisfying the given condition and by those elements returns a new array. Lastly, forEach() loops through the whole array, do it's given task, but doesn't return anything.

4. Arrow function is the shorthand of the function declaration and assigning its returning value in a variable. 

5. When we need to compare between just two conditions in a single line of code, we can do that by using template literals.
